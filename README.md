# Superstore_sales_analysis_SQL
Project Overview

The Superstore Sales Analysis SQL project focuses on analyzing retail sales data to uncover meaningful business insights using SQL. This project uses the Superstore dataset provided by Tableau, which contains information about customers, orders, products, sales, profit, and regions.

The goal of this project is to simulate a real-world business scenario and use SQL to analyze sales performance, customer behavior, and product profitability.


1.Objectives

Analyze overall sales and profit performance

Identify top customers and top-selling products

Find loss-making products and categories

Analyze sales by region, segment, and category

Demonstrate SQL skills for data analyst roles
